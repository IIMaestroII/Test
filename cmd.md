# Подсказки по командной строке

Команда смены директории 
```sh
cd c:\folder_name
```
Команда отображения текущей директории в Linux и MacOS
```sh
pwd
```
Листинг тикущей директории
```sh
dir
```
Linux, MacOS:
```sh
ls
```
Удаление файла в Widows:
```sh
del <filename>
```
Linux MacOS
```sh
rm <filename>
```
